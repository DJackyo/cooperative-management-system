# Directorio para almacenar comprobantes de pagos

Este directorio almacena los archivos de comprobantes de pago (vouchers) cargados por los usuarios.

## Formatos permitidos
- JPG/JPEG
- PNG
- PDF

## Tamaño máximo
- 5MB por archivo

## Nomenclatura
Los archivos se guardan con el formato: `pago-{timestamp}-{random}.{ext}`

Ejemplo: `pago-1704672345678-123456789.pdf`
